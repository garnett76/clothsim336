1. change to clothsim336 directiroy
2. run "npm install"
3. run "npx parcel ./src/index.html"

Controls: x = move a sphere back and forth on the x-axis
          z = move a sphere back and forth on the z-axis
          c = show the invisible boxes used to prevent clipping into model
          f = cycle between capes
          w = toggle wind modifier
          j = move turbine right
          l = move turbine left
          i = move turbine forward
          k = move turbine back
          p = decrease turbine power
          P = increase turbine power
